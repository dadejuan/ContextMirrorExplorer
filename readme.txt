Prokaryotic Co-evolutionary Networks (MICROME project)

This directory contains a script (ContextMirrorExplorer_v1.0.pl) for interrogating the included co-evolutionary networks for lists of query genes.

Subdirectories contain the information required by ContextMirrorExplorer_v1.0.pl

/
	-> data/
		->misc/		contains information about gene and species identifiers
		->networks/	contains files with the results of the co-evolutionary analyses for every prokaryotic species
